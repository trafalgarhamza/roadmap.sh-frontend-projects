# Personal Portfolio Website

A dark-themed Bootstrap portfolio built with plain HTML and Bootstrap utilities. This multi-page site is designed as a personal showcase with responsive layout, modern spacing, and portfolio-focused content.

## Pages

- `homepage.html` — portfolio landing page with a hero introduction and navigation.
- `projects.html` — featured project cards and case study summaries.
- `articles.html` — insights and article-style content for portfolio storytelling.
- `contact.html` — contact form for project inquiries and collaboration.

## Features

- Bootstrap 5 responsive layout and utility-based styling
- Consistent dark portfolio theme using Bootstrap classes
- Clear top navigation across all pages
- Semantic HTML structure for accessibility and readability
- Simple contact form layout ready for backend wiring

## Usage

1. Open the `03-personal-portfolio` folder in your editor or browser.
2. Open `homepage.html` in a web browser.
3. Use the navigation links to explore the portfolio pages.

## Notes

- Styling is provided via the Bootstrap CDN.
- No custom CSS file is required in this folder.
- The site is static HTML-only and works locally without a server.
