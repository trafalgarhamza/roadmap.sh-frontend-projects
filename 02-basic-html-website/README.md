# Basic HTML Website

A small static website built with plain HTML to demonstrate a multi-page site with semantic structure and navigation.

## Pages

- `homepage.html` — landing page with an introduction and navigation links.
- `projects.html` — a simple projects showcase page.
- `articles.html` — an articles list page with example posts.
- `contact.html` — a contact form page for collecting user messages.

## Features

- Semantic HTML5 page structure (`header`, `main`, `section`, `article`, `footer`)
- Responsive viewport metadata for mobile-friendly rendering
- Clear navigation across all pages
- Metadata tags for description, keywords, and author

## Usage

1. Open the `02-basic-html-website` folder in your editor or browser.
2. Open `homepage.html` in a web browser to view the site.
3. Use the navigation links to move between pages.

## Notes

- This is a static website only, with no CSS or JavaScript included.
- All pages are linked using relative paths for easy local browsing.
